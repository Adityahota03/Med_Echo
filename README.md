# 🩺 MedEcho – AI-Powered Healthcare Assistant

MedEcho is an AI-powered healthcare web application designed to provide quick medical assistance, symptom analysis, first-aid guidance, and emergency support. The project aims to make healthcare more accessible through an intuitive and responsive interface.

## 🌐 Live Demo

**Website:** https://adityahota03.github.io/Med_Echo/

---

## 📖 Overview

MedEcho combines artificial intelligence with healthcare services to assist users in understanding symptoms, receiving first-aid recommendations, managing medications, and accessing emergency features. It is designed as a user-friendly platform that promotes faster healthcare support and awareness.
